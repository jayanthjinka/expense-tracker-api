# Expense Tracker Backend

A backend API built with FastAPI for tracking personal expenses.

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Pytest

## Features

- User registration
- JWT authentication
- Create expense
- View expenses
- Update expense
- Delete expense
- Dockerized backend and database

## Run with Docker

docker compose up --build
